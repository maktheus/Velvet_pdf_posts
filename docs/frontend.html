componentes principais

ProductCard
Card de produto reutilizável em grids e carrosséis.
  • productProduct
  • showBadge?boolean
  • onAddToCart?(id) → void
  • variant?'grid' | 'list'
CartDrawer
Gaveta lateral com itens do carrinho, totais e CTA de checkout.
  • openboolean
  • onClose() → void
CheckoutStepper
Indicador de progresso dos 4 passos do checkout.
  • currentStep1 | 2 | 3 | 4
  • stepsStepDef[]
QuizEngine
Motor do quiz com estado de perguntas/respostas e lógica de resultado.
  • questionsQuestion[]
  • onResult(ids) → void
ReviewStars
Exibição de avaliações com estrelas, nota média e distribuição.
  • ratingnumber
  • countnumber
  • distribution?number[5]
FlashSaleTimer
Contador regressivo para promoções relâmpago, atualizado a cada segundo.
  • endsAtDate
  • onExpire?() → void

estado global

Gerenciado via CartContext (React Context API). O carrinho persiste em localStorage e é hidratado no cliente.

addToCart
(product, qty?) → void
Adiciona item ao carrinho ou incrementa quantidade se já existir.
removeItem
(productId) → void
Remove completamente um item do carrinho pelo ID.
setItemQty
(productId, qty) → void
Define a quantidade exata de um item. Qty = 0 remove o item.
cartCount
number (readonly)
Total de itens no carrinho (soma das quantidades). Usado no badge do header.
cartOpen
boolean
Controla a visibilidade do CartDrawer. Alterado com setCartOpen.
cartTotal
number (readonly)
Valor total do carrinho em centavos. Formatar com formatBRL().

Variável Obrigatória Descrição
NEXT_PUBLIC_API_URL ✅ sim URL base da API Spring Boot. Ex: https://api.patadeveludo.com.br
NEXT_PUBLIC_MP_PUBLIC_KEY ✅ sim Chave pública do Mercado Pago para o SDK JS de pagamento no browser
NEXT_PUBLIC_GA_ID ⚪ opcional ID do Google Analytics 4 para rastreamento de eventos
NEXTAUTH_SECRET ✅ produção Secret para assinatura de sessões NextAuth (se utilizado)
